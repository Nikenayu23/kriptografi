Aplikasi Kriptografi Modern (AES)
Aplikasi web sederhana untuk melakukan operasi Enkripsi dan Dekripsi teks menggunakan algoritma kriptografi modern Advanced Encryption Standard (AES).

Dekripsi Teks: Mengembalikan ciphertext kembali menjadi plaintext aslinya.

Antarmuka Sederhana: Antarmuka berbasis web yang simple dan user-friendly untuk mempermudah penggunaan.

Cara Menggunakan
Untuk Enkripsi:

Masukkan teks yang ingin dienkripsi pada kolom "Masukkan teks untuk dienkripsi".

Klik tombol "Enkripsi".

Ciphertext (Hasil Enkripsi) akan ditampilkan di bagian "Hasil Enkripsi".

Untuk Dekripsi:

Salin dan tempel ciphertext (hasil enkripsi) pada kolom "tempel hasil enkripsi di sini".

Klik tombol "Dekripsi".

Plaintext (Hasil Dekripsi) akan ditampilkan di bagian "Hasil Dekripsi".
